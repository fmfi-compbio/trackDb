Description

This track shows regions of the human genome that align to a large number of regions in the mouse genome.

Methods

The alignments were computed with the AVID program, developed by Lior Pachter and Nicolas Bray (Bray N., Fabrikant A., Lord J., Schwartz J., Dubchak I., Pachter L. AVID: A global alignment program for large genomic sequences. In preparation. 2001). Refer to the VISTA website for more information on AVID.
