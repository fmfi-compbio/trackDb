Description

The Ensembl Non-Protein-Coding Genes track shows Ensembl genes that are not protein-coding.

Display Conventions

Different types of genes are color coded differently:

  • miRNA: red
  • misc_RNA: black
  • snRNA: blue
  • snoRNA: magenta
  • rRNA: cyan
  • scRNA: yellow
  • Mt_tRNA: green
  • Mt_rRNA: darker green
  • others: gray

Methods

Ensembl Genes data were downloaded using BioMart and protein-coding genes were filtered out. A track in extended bed 8 format was produced based on these data. The score, thickStart, and thickEnd fields are not used and set to 0.
